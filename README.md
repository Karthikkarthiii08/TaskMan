# 🚀 TaskMan

TaskMan is a modern and responsive task management application built with **React.js**. It helps users organize their daily tasks by allowing them to add, manage, categorize, prioritize, and track task completion through an intuitive user interface.

---

## ✨ Features

- ✅ Add new tasks
- 📝 Manage daily tasks efficiently
- 🎯 Set task priority (High, Medium, Low)
- 📂 Organize tasks by category (General, Work, Personal)
- 📊 Real-time Progress Tracker
- 🗑️ Delete individual tasks
- 🧹 Clear all tasks
- 💾 Local Storage support (tasks persist after refresh)
- 📱 Responsive and modern UI
- 🌙 Attractive dark theme

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Karthikkarthiii08/TaskMan.git
```

Go to the project directory

```bash
cd TaskMan
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
